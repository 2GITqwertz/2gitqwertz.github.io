FAQ

Which devices are supported?

According to the App Store listing, Senet Reborn is available for iPhone and iPad (iOS 26.0 or later; Apple also lists iPod touch in compatibility information).

Can I play against AI?

Yes. The store description mentions an adaptive AI opponent and three difficulty levels (Easy, Medium, Hard).

Is there a local multiplayer mode?

Yes. The App Store description includes local 2-player multiplayer on one device.

Which rule modes are available?

Classic Mode and Flow Mode. Detailed rules are documented on the HowToPlay page.

Reporting an Issue

If something does not work, this structure helps speed up troubleshooting:

  • Device model and iOS version
  • What you expected vs. what happened
  • Whether the issue can be reproduced
  • Which mode was active (Classic or Flow)
  • Optional screenshot of the game state
